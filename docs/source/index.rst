Welcome to xdispersion's documentation!
===================================

.. image:: https://raw.githubusercontent.com/miniufo/xdispersion/master/pics/illustration.png

This is a `Python` package for analyzing relative dispersion behaviors of Lagrangian particle
pairs.  This is a classical problem of turbulence and geophysical fluid dynamics
(GFD) originated from Richardson (1926).  This project is
`published on GitHub <https://github.com/miniufo/xdispersion>`__.
